# Kawasaki Sales Performance Analysis using Python

## 📌 Overview

This project focuses on analyzing **Kawasaki bike sales** using Exploratory Data Analysis (EDA). Rather than studying the entire market, this analysis examines the sales performance of a single brand to uncover product trends, pricing patterns, and customer preferences.

The project demonstrates how business insights can be generated using Python, Pandas, and data visualization.

---

## 🎯 Objectives

* Analyze Kawasaki's sales performance.
* Compare sales across different bike models.
* Study regional sales trends.
* Understand customer ownership patterns.
* Explore relationships between selling price and resale price.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## 📊 Analysis Performed

* Data Cleaning & Exploration
* Model-wise Total Sales Revenue
* Model-wise Average Selling Price
* State-wise Sales Performance
* State-wise Average Selling Price
* Manufacturing Year Analysis
* Registration Year Analysis
* Owner Type Distribution
* Fuel Type Distribution
* Seller Type Analysis
* Price vs Resale Price Analysis (Scatter Plot)
* Mileage Analysis

---

## 📈 Key Insights

* Identified the highest revenue-generating Kawasaki model.
* Compared average selling prices across models.
* Evaluated sales performance across different states.
* Studied ownership trends in the resale market.
* Explored the relationship between resale value and original selling price.
* Examined mileage patterns across Kawasaki models.
